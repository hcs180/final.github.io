# 期末專案網站

## 網站內容
- `index.html`：首頁與網站亮點
- `about.html`：關於我、技能地圖、學習方法
- `projects.html`：作品集、搜尋、分類篩選、收藏、作品細節彈窗
- `project-iot.html`：代表專題「智慧物聯共享站」完整說明
- `experience.html`：競賽、證照、實習、活動展示
- `articles.html`：學習反思文章與閱讀模式
- `contact.html`：聯絡資訊、表單驗證、草稿保存示範
- `style.css`：網站樣式與 RWD
- `script.js`：互動功能
- `images/`：網站圖片資料夾

## JavaScript 功能

1. 深色模式切換，並用 localStorage 記住設定
2. 目前分頁自動標記 active
3. 首頁按鈕使用 JS 跳轉分頁
4. Scroll Reveal 滾動動畫
5. 數字統計動畫
6. 作品分類篩選
7. 作品關鍵字搜尋
8. 作品收藏，並用 localStorage 保存
9. 作品細節 Modal 彈窗
10. 代表專題技術模組切換
11. 文章頁閱讀模式切換
12. 聯絡表單驗證
13. 聯絡表單草稿保存與清除
14. 回到頂部按鈕
15. 手機版導覽列點擊後自動收合
